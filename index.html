A Peculiar Washing Machine

A washing machine is a device used to thoroughly clean your clothes with detergent and water by cycling clothes for a period of time. A washing machine is typically paired with a dryer. For this project, I am specifically using the shared laundry machine in my apartment complex. This particular washing machine has an opening on the top that allows you to insert your clothes by pulling up a handle. To control the machine, the washer has three knobs. One where you select the size of the load, one to select the temperature of the water, and finally one to select how long you want the washer to run for. To start the machine you press the knob that controls the time. To stop the machine, either wait for the time to run out or open the lid.

The Investigation

To get a better understanding of how people interact with our washing machine, we will now observe users and conduct interviews afterwards. There answers are summarized below!

Observations
  • Users very frequently turn the knobs too far, and they have to spin them around again
  • Users open the lid and put in clothes quickly with ease
  • Users never know how long the laundry machine will run for
  • Users seem to have neutral to negative demeanors when using the machine

“What was your initial reaction to the load size and the temperature knobs?”
  • Users seemed to not think much about these knobs. The temperature knob clearly was to determine the temperature of the water, however users did not know exactly what the load size knob would do, but this did not bother them.

“What was your initial reaction to the control knob on the right?”
  • This knob greatly confused users. The knob has numbers that all users collectively agreed they thought would represent the length of the wash. However, this was never the case and the machine would run much longer. Some users speculated that the number could interact with the load size knob.

“Can you describe any difficulties, if any, you faced while using the washing machine?”
  • Small difficulties users faced were over-turning the knobs, but this was an easy mistake to fix. One user forgot to use detergent. The difficulty that every user reported was not knowing how to use the knob on the right.

“Did you find the washing machine to be intuitive to use?"
  • Most users agreed that they understood everything about the washing machine except for how to set the time properly.

“What was your reason for using the washing machine?”
  • Answers varied slightly here, but most people responded they were either out of clean clothes or that this is a weekly chore for them.

Personas

Based off of our investigation, we will now create some personas! The personas will create more personal characters to represent the users we have observed.

Pressured Patty is a busy college student working two jobs, who wants to get her laundry done in the most efficient way possible so she can work on other tasks.

Patty faces the issues of not understanding how the time works on the machine, turning the knobs too far, and being confused by the options. I have Patty representing the user doing this as a chore when they would rather be doing more important things, therefore they do not want to take the time to figure out the interface.

Tired Tyler is an exhausted college student who stayed up all night working on an assignment. He wanted to go to bed until he realized he had no clean clothes left, so now he wants to quickly do laundry and then go to bed.

Tyler faces the issue of not understanding how the time works on the machine and not knowing exactly what each knob does. I have Tyler representing users who have run out of clean clothes and want to quickly get their laundry done.
